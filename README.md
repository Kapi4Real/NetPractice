# NetPractice

## Description
Projet 42 pour s'entraîner aux bases des réseaux informatiques : IP, masques, routage, passerelles, routeurs, switches, modèle OSI.

## Instructions
- Lancer l'interface NetPractice (ou `./run.sh` si fourni)
- Résoudre chaque niveau
- Exporter chaque configuration en `.json`
- À rendre : 10 fichiers (`level1.json` à `level10.json`) + ce README

## Resources
- [RFC 791 – Internet Protocol (IP)](https://datatracker.ietf.org/doc/html/rfc791)
- [RFC 950 – Subnetting](https://datatracker.ietf.org/doc/html/rfc950)
- [Cisco Networking Basics](https://www.cisco.com/c/en/us/solutions/enterprise-networks/what-is-networking.html)
- [OSI Model](https://fr.wikipedia.org/wiki/Mod%C3%A8le_OSI)

**Concepts étudiés :** TCP/IP, sous-réseaux, passerelles, routeurs, switches, OSI.
